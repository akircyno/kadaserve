# Source Notes

Fetched from `https://github.com/karanb192/awesome-claude-skills`.

Commit checked: `18f038311f1041c92158357692bf8ddfa8c5da4d`.

The upstream repository is an awesome-list catalogue. It contains `README.md`, `CONTRIBUTING.md`, `LICENSE`, and GitHub metadata, but no root `SKILL.md`. Because of that, it cannot be installed as one direct runnable skill. The local `SKILL.md` in this folder is a KadaServe project wrapper that turns the catalogue into a usable planning reference.

Use it to choose future skills for:

- Complex logic design.
- Testing and quality workflows.
- Systematic debugging.
- Architecture and refactoring.
- Data analysis and analytics planning.
- Thesis-focused intelligent-system framing.

For actual skill installation later, inspect the specific linked source repository from the catalogue and verify that it contains a valid `SKILL.md`.
